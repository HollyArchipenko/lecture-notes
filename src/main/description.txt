Данный сервис создается с целью восполнения пробелов в знаниях, а также для более полного изучения математических предметов.
    Возможности, которые предоставляет данный сервис:
        1) Загрузка лекций.
        2) Подписка на курс лекций.
    Группы пользователей:
        1) guest  может загружать лекции с сервера. Не имеет профиля и, соответственно, не может подписываться на лекции.
        2) user  может как загружать лекции с сервера, так и подписываться на них. Чтобы гость мог стать пользователем, он должен пройти регистрацию и заполнить обязательные поля профиля. Пользователь может изменять свой профиль.
        3) administrator может загружать лекции как с сервера, так и на сервер. В ближайшее время данный статус будет только у разработчиков.
